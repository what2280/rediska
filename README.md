# rediska
rediska only
Редактор консоли и менеджер компьютера.

Работает только на ЛИНУКС на Термукс НЕ РАБОТАЕТ, т.к. нет папки home!

установка:

apt update

apt upgrade

apt install zip

apt install unzip

apt install git

git clone https://github.com/what2280/rediska

cd rediska

unzip rediska.zip

python3 installer.py
