# rediska
rediska only
редактор консоли и менеджер компьютера.
работает только на ЛИНУКС и ТЕРМУКС!

установка:

apt update

apt upgrade

apt install zip

apt install git

git clone https://github.com/what2280/rediska

cd rediska

unzip rediska.zip

python3 installer.py
