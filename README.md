# rediska
rediska only
редактор консоли и менеджер компьютера.
работает только на ЛИНУКС на термукс НЕ РАБОТАЕТ т.к. нету файла home!

установка:

apt update

apt upgrade

apt install zip

apt install unzip

apt install git

git clone https://github.com/what2280/rediska

cd rediska

unzip rediska.zip

python3 installer.py
